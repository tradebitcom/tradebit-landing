FAQ

Часті запитання

Яка різниця між v1 та v2?
v1 — це інструмент для одного користувача без монетизації. v2 — мультикористувацька платформа з оплатою через Telegram Stars, SQLite, багатомовним інтерфейсом і логікою для бізнесу. Якщо потрібен продукт для заробітку — обирайте v2.
Чи можна запускати v1 на будь-якому сервері?
Так. Підійде будь-який Linux VPS з Python 3.10+. Навіть безкоштовний тариф AWS EC2 може бути достатнім.
Чи потрібні API-ключі бірж?
Для публічних даних (тікери, ордербуки) ключі не потрібні. Вони потрібні лише для приватних endpoint (наприклад, комісії на вивід).
Які біржі підтримуються?
Усі 12: Binance, Bybit, OKX, Gate.io, Bitget, KuCoin, Huobi, MEXC, BingX, HitBTC, Poloniex, Bitfinex.
Чи можна перейти з v1 на v2?
Так — у v2 є утиліта міграції, яка один раз переносить ваші налаштування в нову SQLite-базу.
Інтерфейс англійською?
v1 має лише російський інтерфейс. Для англійської + інших мов використовуйте v2.
